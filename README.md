# Wireshark Traffic Analysis & TLS Decryption Lab

## 📌 Overview
This project demonstrates hands-on experience in analyzing encrypted network traffic using Wireshark. The lab focused on decrypting TLS traffic, inspecting HTTP/2 packets, reconstructing fragmented data, and extracting hidden information.

---

## 🛠️ Tools Used
- Wireshark
- TLS Key Log File
- HTTP/2 Protocol Analysis

---

## 🔍 Key Skills Demonstrated
- TLS traffic decryption using pre-master secret log file
- HTTP/2 packet inspection and header analysis
- Identification of authority headers
- Reassembly of fragmented packet data
- Extraction of hidden data from compressed responses
- Hex-to-ASCII decoding for payload analysis

---

## 📊 Tasks Completed
- Identified HTTP/2 authority header:
  - `safebrowsing[.]googleapis[.]com`
- Decrypted encrypted traffic using key log file
- Analyzed HTTP/2 streams and DATA packets
- Reassembled fragmented payloads
- Extracted hidden encoded data from network traffic

---

## 🧠 Key Learning Outcome
This lab strengthened my ability to:
- Analyze encrypted traffic in real-world scenarios
- Navigate Wireshark efficiently under pressure
- Understand how attackers may hide data within legitimate traffic
- Perform deep packet inspection and reconstruction
